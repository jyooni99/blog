# 테스트 MDX 포스트

이것은 MDX 파일 테스트용 포스트입니다.

## 특징

MDX를 사용하면 마크다운과 React 컴포넌트를 함께 사용할 수 있습니다.

### 코드 블록 예제

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}

console.log(greet("MDX"));
```

```typescript
interface Post {
  title: string;
  content: string;
  date: Date;
}

const post: Post = {
  title: "테스트 포스트",
  content: "MDX 테스트 중",
  date: new Date(),
};
```
